Test Data
=========

This folder contains test data obtain from different sources:

* ``ytk.tsv.xz`` is a compressed CSV version of the ``YTK_Parts.xls`` file that
  comes with the YTK GenBank files.

  of a multigene assembly, provided by `Sebasti√°n Sosa Carrillo <https://research.pasteur.fr/en/member/sebastian-sosa-carrillo/>`_
